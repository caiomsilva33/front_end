Front End: O front-end pode ser definido como a profissão que cuida da apresentação de um site. Ou seja, cuida das partes visuais e da interação entre usuários e tela. Gerencia a usabilidade, o design e a experiência do usuário.

Back End:. É o código que conecta a internet com o banco de dados, gerencia as conexões dos usuários e alimenta a aplicação web. O backend trabalha em conjunto com o frontend para entregar o produto para o usuário final.

Banco de Dados: Um banco de dados é uma coleção organizada de informações - ou dados - estruturadas, normalmente armazenadas eletronicamente em um sistema de computador. Um banco de dados é geralmente controlado por um sistema de gerenciamento de banco de dados (DBMS).
